# Node.js Starter

## Overview
A simple Node.js starter project using Express.js framework. This project provides a basic web server with a welcome page and API endpoint.

## Project Structure
- `index.js` - Main application file with Express server
- `package.json` - Project dependencies and scripts

## Features
- Express.js web server
- Homepage with styled UI
- Sample API endpoint at `/api/hello`
- Configured to run on port 5000

## Getting Started
The server starts automatically. Visit the webview to see your application running.

## API Endpoints
- `GET /` - Homepage
- `GET /api/hello` - Returns a JSON response with a greeting message
