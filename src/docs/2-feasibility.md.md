# 2 — Feasibility (Реалистичность)

• Источники минутной/15-минутной погоды ограничены.
• Open-Meteo даёт часовые данные — можно интерполировать.
• Apple Weather / Meteoblue дают high-resolution модели (через unofficial API).
• Вариант: комбинировать два источника и вычислять локальный диапазон.
• Радиус 1–3 км реалистичен: достаточно 3 точек семплирования по кругу.
