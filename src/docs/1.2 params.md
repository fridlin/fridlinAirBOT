# Weather Parameters (P1–P9)
# Unified list for all features of FridlinAir

P1. Temperature  
P2. Feels-like temperature  
P3. Humidity  
P4. Wind  
    - speed  
    - gusts  

P5. Precipitation  
    - presence  
    - intensity  

P6. UV index  

P7. Short-term changes (next 2–3 hours)  

P8. Hamsin-effect  
    - low humidity  
    - dry strong wind  
    - dryness in eyes/mouth  
    - not dependent on high temperature  

P9. Comparison with 24h ago  
    - temperature  
    - feels-like  
    - humidity  
    - wind speed  
    - wind gusts  
    - precipitation
