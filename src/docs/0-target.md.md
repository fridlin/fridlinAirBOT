## Purpose
Этот файл фиксирует главную цель проекта FridlinAir: какую боль клиента мы решаем и зачем этот сервис вообще нужен.

FridlinAir решает главную проблему погоды в Израиле: условия меняются на коротких расстояниях и по часам. Обычные приложения дают общий усреднённый прогноз, который не помогает человеку, который выходит из дома, едет 5–15 км и возвращается.

Цель FridlinAir — показать точную локальную погоду вокруг человека (1–3 км) и по маршруту, с интервалами 15–30 минут: что надеть, что взять с собой, будет ли дождь, ветер или резкие перепады.
